
# Rhythm Loops Directory

This directory contains the audio files for the Loop Station application.

## Folder Structure
To add a new rhythm, create a new folder here with the rhythm's name (e.g., `MPB`, `ROCK`) and include the following exactly named files:
- `intro.mp3`
- `virada1.mp3`
- `virada2.mp3`
- `virada3.mp3`
- `final.mp3`
- `prato.mp3`

Supported formats: `.mp3`, `.wav`, `.ogg`.

### Example
/public/loops/
  /MPB/
    intro.mp3
    virada1.mp3
    virada2.mp3
    virada3.mp3
    final.mp3
    prato.mp3
  /ROCK/
    ...
